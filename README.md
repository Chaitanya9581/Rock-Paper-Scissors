# 🎮 Rock Paper Scissors - Flask Web Game

A simple Rock Paper Scissors game built using **Python** and **Flask**.  
The game runs on **localhost** and allows users to play against the computer in a web browser.

---

## 🚀 Features

- Web-based interface
- Play against computer
- Random computer choice
- Instant result display
- Beginner-friendly project structure

---

## 🛠 Technologies Used

- Python 3.x
- Flask
- HTML (Jinja2 templating)

## 🎯 How to Play

1. Click **Rock**, **Paper**, or **Scissors**
2. The computer randomly selects its choice
3. The result is displayed instantly
4. Play as many times as you like!

---

## 🧠 Game Logic

- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice = Tie
